Tutorial 4: Objects Part 2: Prototypal Inheritance and `this`


Prototypal Inheritance

To add a new property to a constructor you have to add it to the constructor function Javascript objects will inherit properties and methods from a protoype.

Lets say we have this function

  function Dog(name, breed, furColor)
  {
      this.fName = name;
      this.breed = breed;
      this.furColor = furColor;
  }

If we wanted to add the dog's age we will use the prototype property

    Dog.prototype.age = "Two";

`this`

The keyword this refers to the object it is associated with. If it is used in a function then it refers to a global object. If it is used in a method then it refers to the owner object.

this in a method

   var dog = {
       firstName: "Lolly",
       lastName: "Pop",
       breed: "Maltese Yorkie Mix",
       fullName : function(){
           return this.firstName + " " + this.lastName;
       }
   }

It will return as- Lolly Pop



this in a function

    function exampleFunction()
    {
        return this;
    }

In this code this is refering to a global object. If you were to use this same function but in strict mode, it would showcase "undefined" rather than "[object Window]"
