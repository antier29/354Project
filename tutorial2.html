Functions


Named Functions

In JavaScript, functions are defined by using the keyword function. 

      function subtract(a,b)
      {
        return a - b;
      }

In this function we want to subtract a and b. But with function declarations, like we have above, they are executed later. You have to call to it later. So to actually use the function we would simply use the function name follwed by the two numbers we want to subtract. 

      subtract(5,1);
If you were to run this, you would get an answer of 4.

Arrow Functions

Also known as "fat arrow" functions, this type of function is a shorter way of writing a normal function. They can't be used as constructors though. 

  const multiply(a,b) =>
  {
    return a * b
  };

One cool thing is the curly brackets are not needed if there is only one expression present.

Scope

Scope determines the visibility of variables. For local scope, only the variable inside the function can be seen. If a variable is outside the function, it can not be seen

  //The code here can not use/see favColor

  function colors()
  {
    var favColor = "Blue";

    //The rest of the code in here can use favColor
  }

For global scope all functions can use the variable

    var favColor = "Blue";
    //any code here can use/see favColor

    function colors()
    {
      //you can use/see favColor here.
    }
