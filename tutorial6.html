REST stands for REpresentational State Transfer. It is a design patter for API's. It has 6 constraints in order for interfaces to be restful. 1. Uniform interface 2. Client-Sever 3. Stateless 4. Layered system 5. Cacheable 6. Code on demand When an API is restful its when a server will show(transfer) the state of what you requested. Let's say you sign up for Instagram or Twitter. Postman is a software that is for API development. This is used between developers to make, test and adjust API's. This software makes API advancement easy and efficient.
