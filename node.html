Node, What Is It?


Node was created in 2009 by Ryan Dahl. It was built to give developers the ability to have JavaScript code run in computer process rather than a browser. Node is created on V8 engine. It is written in 3 different programming languages. JavaScript, C++ and C.
V8 is what runs Google Chrome. It is an open source engine that runs Javascript in a browser. V8 is the domain that Javascript executes in. Node.js is now widely used and V8 helped its success.
